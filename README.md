# repositorio-forum

O que busco aprender aqui

Minha meta nesta disciplina é ganhar confiança no básico: escrever código com organização, entender como funciona o Git e o GitHub, criar meus próprios arquivos e atualizar tudo com tranquilidade. Quero aprender a registrar meu progresso, revisar minhas atividades e trabalhar melhor com as ferramentas que os programadores usam no dia a dia.

Por que programação

Me interesso por programação porque ela abre caminhos. Gosto da sensação de construir algo que funciona, entender o que acontece por trás da tela e imaginar como isso pode ajudar outras pessoas, principalmente na educação.
